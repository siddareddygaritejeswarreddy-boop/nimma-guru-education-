# 📚 Nimma Guru Education

Nimma Guru Education is an educational platform designed to help students connect with teachers easily and improve their learning experience. The application provides subject-wise teacher details, teacher availability timings, student management, and secure login functionality.

---

## 🚀 Features

- 🔐 User Login & Authentication
  - Email & Password Login
  - Google Sign-In Support

- 👨‍🏫 Teacher Management
  - Subject-wise Teachers
  - Teacher Contact Information
  - Teacher Available Timings

- 👨‍🎓 Student Database
  - Student Information Storage
  - Easy Access to Student Records

- 📘 Subject Management
  - Multiple Teachers for Each Subject
  - Organized Educational Structure

- 📱 User-Friendly Interface
  - Clean and Simple UI
  - Easy Navigation for Students

---

## 🛠️ Technologies Used

- Android Studio
- Java / Kotlin
- Firebase Authentication
- Firebase Realtime Database / Firestore
- XML for UI Design
